    Elements

    Key elements used in the design of the website

    Element Description
    additional resource

    Buttons

    Includes styles from both Bootstrap Material Design and Internal CSS

    https://mdbootstrap.github.io/bootstrap-material-design/docs/4.0/bootstrap-components/buttons/

    Filter Forms

    Included on homepage. Filters course results by either keyword search or category selection.

    https://mdbootstrap.github.io/bootstrap-material-design/docs/4.0/bootstrap-components/forms/

    Course Title

    Breif course description pulled from database.

    Course Card Tile

    Included on homepage. Each course title is generated using JavaScript. The card data is stored in JSON (JavaScript Object Notation) object.

    This includes the image, title, description, and link to course page.

    https://getbootstrap.com/docs/5.0/components/card/

    Embedded Video

    The embeded video is loaded from the JSON (JavaScript Object Notation) database using JavaScript functionality. The CSS applied is custom internal css. The video is loaded using YouTubes embed feature.

    https://support.google.com/youtube/answer/171780?hl=en

    Video Links with Checkbox

    The course videos are auto generated using JavaScript with JSON (JavaScript Object Notation). This is combination of a <li> element </li> and checkboxes.

    https://mdbootstrap.github.io/bootstrap-material-design/docs/4.0/material-design/selections/#checkboxes
